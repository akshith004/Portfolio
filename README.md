# 🌐 Personal Portfolio Website

This is my **portfolio website** showcasing my profile, skills, projects, and contact details. Built with **HTML, CSS, and JavaScript**, it's responsive and styled with modern UI effects.

## ✨ Features
- Smooth scrolling navigation
- Responsive mobile menu
- Skills and project sections
- Contact details with links
- FinisherHeader animated background

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)
- FinisherHeader Animation
- Google Fonts & Font Awesome
